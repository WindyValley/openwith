# 说明
本工具只在提供一个通用的文件超链接协议handle，当打开本链接时，可以通过预先配置好的文件编辑器来打开超链接指示的文件，并调转到指定行。
# 背景
- 文本搜索工具ripgrep(rg)支持为输出结果打上超链接标记，在支持超链接的终端（如Windows Terminal、Wezterm）上，可以通过点击超链接来打开文件，而不用手动执输入命令或者在文件管理中找到文件再打开
- file协议不支持指定行列号，而VSC通过vscode协议的超链接支持这种能力，但是想VSC这种有自定义超链接协议的编辑器又只是少数，有时候我可能想用vim或者Kate等
在这样的前提下，制作本工具的想法应运而生
